# Il Mondo delle Quattro Ruote — Automotive Educational Platform

Welcome to the repository for **Il Mondo delle Quattro Ruote** (The World of Four Wheels), an educational and informative web platform dedicated to the automotive industry, engine technologies, and iconic car manufacturers. 

The live website is available via GitHub Pages at [mascoloandrea.github.io/Il-Mondo-delle-Quattro-Ruote/](https://mascoloandrea.github.io/Il-Mondo-delle-Quattro-Ruote/).

---

## 🏎️ Project Overview

This project serves as an introductory web development showcase focused on the automotive world. It combines curated technical documentation about internal combustion and electric engines with dedicated structural profiles of world-renowned luxury and performance car brands.

### Key Content & Sections:
- **Engine Technical Insights:** Detailed breakdowns of engine types, including:
  - Electric Motors
  - Diesel Engines
  - Two-Stroke Engines
  - Hydrogen Vehicles
- **The Internal Combustion Engine (ICE) Deep Dive:** A comprehensive, step-by-step analysis of the **4-stroke engine cycle**:
  1. *Aspiration (Intake)*
  2. *Compression*
  3. *Combustion & Expansion*
  4. *Exhaust*
- **Market Data Visualization:** Integrated statistical tables highlighting automotive fuel-type market shares.
- **Iconic Manufacturer Showcases:** Dedicated pages for legendary car brands:
  - Ferrari
  - Lamborghini
  - Mercedes-Benz
  - Bugatti

## 🛠️ Built With

The website is crafted using lightweight, fundamental frontend technologies, serving as an excellent example of semantic structure and clean styling.

- **HTML5:** Semantic architecture, structured tables, and deep linking for seamless reading navigation.
- **CSS3:** Custom styles, typographic hierarchies, and layout structures tailored to an editorial automotive theme.
- **Hosting:** GitHub Pages
